# MyCSharpNotebook

A personal C# learning journal where I reflect on my programming journey.

## Why did I make this repo?

* To keep a record of my C# learning progress.
* To experiment with C# concepts like OOP, recursion, delegates, and data structures.
* To compare C# with programming languages I knew before (Python, C++, Java, etc.).
* To have a reference for myself learning C#.

## Contents

* **Notes:** Observations and insights while learning C#.
* **Code snippets:** Examples of C# syntax, algorithms, and patterns.
* **Projects:** Small programs to test and solidify concepts.
* **Reflections:** Comparisons between C# and languages I knew before.
